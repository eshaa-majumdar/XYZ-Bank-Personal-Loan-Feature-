# XYZ Bank Personal Loan Feature – BA Portfolio Project

This repository contains a Business Requirement Document (BRD) and Functional Requirement Document (FRD) for a Personal Loan feature in a mobile banking application.

The requirements were gathered through an AI-assisted JAD (Joint Application Design) session, where AI personas acted as:
- Rohan Mehta – Product Manager (SME)
- Priya Nair – Mobile App Lead Developer

The project demonstrates requirement elicitation, business analysis, functional specification writing, business rules documentation, error handling, and traceability mapping.
